# Levantar el proyecto en local

1. Clonar repo
2. Ejecutar

```
yarn install
```

3. Instalar Nest CLI

```
npm i -g @nestjs/cli
```

4. Levantar contenedor de base de datos

```
docker-compose up -d
```

5. Ejecutar la app

```
yarn start:dev
```

## Stack

- MongoDB
- Nestjs
